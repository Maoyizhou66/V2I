# 面向试车场多工况场景的车路协同V2I融合：基于卡尔曼滤波与高效注意力机制的联合状态估计
决策级融合，卡尔曼滤波与轻量化Transformer联合方法
# 前言
自动驾驶技术尽管在近年来取得了长足进步，但其在全局态势感知、远程感知和盲区感知等方面仍存在显著局限，构成当前系统安全性的关键瓶颈。随着路测基础设施的逐步部署与性能提升，融合车载传感器与路端感知单元的多源信息，能够有效弥补单车智能的不足，包括提供超视距感知范围，消除盲区，提供精确的车道级目标与运动态势信息。因此，车路协同V2I正逐渐成为自动驾驶演进的重要方向，而封闭/半封闭试车场、港口、矿区等限定场景成为该技术率先实现工程化落地的主要应用场景。本文聚焦于比亚迪南宁反季节试验场（亚洲唯一此类专用场地）这一典型限定场景，从相机部署方式与观测视角出发，将纯视觉BEV感知系统划分为三个层级：车端单视角感知、路端固定视角感知，以及全局视角感知。三路感知信号最终在车端进行决策级融合，从而形成车路协同感知下的统一态势认知，为安全策略的制定与执行提供可靠支撑。
