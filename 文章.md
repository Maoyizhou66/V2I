# 面向试车场多工况场景的车路协同V2I融合技术：基于卡尔曼滤波与高效注意力机制的联合状态估计
决策级融合，卡尔曼滤波与轻量化Transformer联合方法
# 前言
自动驾驶技术尽管在近年来取得了长足进步，但其在全局态势感知、远程感知和盲区感知等方面仍存在显著局限，构成当前系统安全性的关键瓶颈。随着路测基础设施的逐步部署与性能提升，融合车载传感器与路端感知单元的多源信息，能够有效弥补单车智能的不足，包括提供超视距感知范围，消除盲区，提供精确的车道级目标与运动态势信息。因此，车路协同V2I正逐渐成为自动驾驶演进的重要方向，而封闭/半封闭试车场、港口、矿区等限定场景成为该技术率先实现工程化落地的主要应用场景。南宁试车场的反季节低温耐久路试中，高速封闭路段的感知距离不足，加上环路本身视野狭窄，两个问题叠加，导致基础碰撞安全存在明显短板。本文聚焦于比亚迪南宁反季节试验场（亚洲唯一此类专用场地）这一典型限定场景，从相机部署方式与观测视角出发，将纯视觉BEV感知系统划分为三个层级：车端单视角感知、路端固定视角感知，以及全局视角感知。三路感知信号最终在车端进行决策级融合，从而形成车路协同感知下的统一态势认知，为安全策略的制定与执行提供可靠支撑。
# 研究现状
车路协同感知的数据粒度和通信带宽之间存在平衡，现有方案分为三个层级。数据级协同，智能网联汽车从其他智能体处接收原始感知数据并执行时空校准，然后进行多源数据的融合。该协同策略能获取高保真信息，从根源上解决远距离感知和盲区遮挡问题，但伴随着海量的数据传输和精确同步的硬性要求，在实际工程化中难度很大。特征级协同，各智能体将原始感知数据经本地编码网络提取为特征向量再进行传输，在智能网联汽车端进行特征融合和推理。该策略显著降低带宽需求，有利于实时性保证，但是特征编码会不可避免引入信息损失和噪声，且各智能体的模型会带来特征空间不匹配的问题，给融合算法设计带来显著挑战。决策级协同，仅共享各智能体独立的感知结果(如目标检测框、分类标签或轨迹)，智能网联汽车端进行结果级融合。该策略通信带宽被降至最低，且各智能体解耦，适合工程落地，但是观测信息的细粒度在逐级压缩中逐渐丢失，且性能高度依赖各智能体的独立能力。综上，
